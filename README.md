# DictionaryX Chrome Extension

**DictionaryX** is a powerful Chrome extension that provides instant definitions for any selected text on a webpage. Supporting over **160 languages** and delivering definitions in **21 languages** using NLP, it’s designed to enhance language accessibility and make web browsing more informative.

## Key Features

- **Multi-language Support**: Detects text in **160 languages** and provides definitions in **21 languages**.
- **AI-powered Definitions**: Delivers accurate definitions powered by AI for better understanding.
- **User-Friendly**: Just double-click on any word or phrase to get a definition without leaving the webpage.
- **Customizable Language Options**: Choose the preferred language for definitions from the options menu.


## Installation

1. Download the extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/dictionaryx-ai%E2%9C%A8/gpngbbekhfmibfhghpedjeanncjejcne).
2. Click **"Add to Chrome"**.
3. Confirm the installation, and you’re ready to go!

## Usage

1. Once installed, simply double-click on any text within a webpage.
2. A pop-up will appear, displaying the definition in the selected language.
3. Customize the language settings by accessing the extension options from the Chrome toolbar.

## Technology Stack

- **JavaScript, HTML, CSS, Shadow DOM / ReactJs**: Core technologies for extension development.
- **AI APIs**: Utilized for providing real-time definitions.
